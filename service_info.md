# 은혜로교회 글램핑 신청 서비스 운영 가이드 (Telegram 버전)

본 가이드는 텔레그램 봇과 구글 캘린더를 활용하여 글램핑 신청을 자동화하는 시스템의 설정 및 운영 방법을 안내합니다.

## 1. 하이레벨 아키텍처
1.  **신청 페이지**: 사용자가 웹브라우저에서 정보 입력 (`index.html`)
2.  **백엔드 처리**: 입력 데이터를 Google Apps Script(`backend.gs`)로 전송
3.  **실시간 알림**: 관리자에게 텔레그램 메시지로 신청 내역 및 [승인] 버튼 전송
4.  **자동 기록**: 승인 버튼 클릭 시 **구글 캘린더**에 일정 자동 등록

## 2. 관리자 설정 방법 (중요)

### A. 텔레그램 봇 생성
1.  텔레그램에서 `@BotFather`를 검색하여 대화를 시작합니다.
2.  `/newbot` 명령어를 입력하고 이름과 사용자 아이디(username)를 설정합니다.
3.  발급된 **HTTP API Token**을 복사하여 `backend.gs`의 `TELEGRAM_BOT_TOKEN`에 넣습니다.
4.  텔레그램에서 `@userinfobot`을 검색하여 자신의 **ID (채팅 ID)**를 확인한 뒤 `backend.gs`의 `TELEGRAM_CHAT_ID`에 넣습니다.

### B. Google Apps Script (GAS) 배포
1.  `backend.gs` 코드를 [Google Apps Script](https://script.google.com/) 프로젝트에 붙여넣습니다.
2.  상단 '배포' > '새 배포' > '웹 앱'을 선택합니다.
3.  **액세스 권한을 '모든 사람(Anyone)'**으로 설정하고 배포합니다.
4.  생성된 **웹 앱 URL**을 `backend.gs` 상단의 `WEB_APP_URL` 변수와 `main.js`의 `SCRIPT_URL`에 입력합니다.
5.  URL을 수정한 후 다시 '배포' > '배포 관리' > '수정' > '새 버전'으로 재배포해야 변경 사항이 적용됩니다.
